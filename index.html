The TableBank Dataset

The Dataset

TableBank is a new image-based table detection and recognition dataset built with novel weak supervision from Word and Latex documents on the internet, contains 417K high-quality labeled tables.

Introduction

The TableBank dataset totally consists of 417,234 high quality labeled tables as well as their original documents in a variety of domains.

The following paper introduces the corpus in detail. If you use the corpus in published work, please cite it:

Minghao Li, Lei Cui, Shaohan Huang, Furu Wei, Ming Zhou, and Zhoujun Li. 2019. TableBank: Table Benchmark for Image-based Table Detection and Recognition. [pdf]

Statistics of TableBank

Task Word Latex Word+Latex
Table detection 163,417 253,817 417,234
Table structure recognition 56,866 88,597 145,463

Get Data

Table Detection part is provided by COCO format while table structure recognition part is distributed in both image files and corresponding txt files. Here is the github repository with a readme:

Published results

The following table reflects the performance on the two part dataset. For table detection, we calculate the precision, recall and F1 in the same way as in [Gilani et al., 2017], where the metrics for all documents are computed by summing up the area of overlap, prediction and ground truth. For table structure recognition, we use the 4-gram BLEU score as the evaluation metric with a single reference.
